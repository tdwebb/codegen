# @appgen/codegen-sandbox

codegen-sandbox package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-sandbox
```

## Usage

```typescript
import { } from '@appgen/codegen-sandbox';
```

## Development

```bash
pnpm -F @appgen/codegen-sandbox build
pnpm -F @appgen/codegen-sandbox test
```
