# @appgen/codegen-git-integration

codegen-git-integration package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-git-integration
```

## Usage

```typescript
import { } from '@appgen/codegen-git-integration';
```

## Development

```bash
pnpm -F @appgen/codegen-git-integration build
pnpm -F @appgen/codegen-git-integration test
```
