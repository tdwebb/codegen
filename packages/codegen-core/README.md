# @appgen/codegen-core

codegen-core package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-core
```

## Usage

```typescript
import { } from '@appgen/codegen-core';
```

## Development

```bash
pnpm -F @appgen/codegen-core build
pnpm -F @appgen/codegen-core test
```
