# @appgen/codegen-artifact-store

codegen-artifact-store package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-artifact-store
```

## Usage

```typescript
import { } from '@appgen/codegen-artifact-store';
```

## Development

```bash
pnpm -F @appgen/codegen-artifact-store build
pnpm -F @appgen/codegen-artifact-store test
```
