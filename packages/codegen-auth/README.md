# @appgen/codegen-auth

codegen-auth package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-auth
```

## Usage

```typescript
import { } from '@appgen/codegen-auth';
```

## Development

```bash
pnpm -F @appgen/codegen-auth build
pnpm -F @appgen/codegen-auth test
```
