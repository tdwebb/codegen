# @appgen/codegen-provenance

codegen-provenance package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-provenance
```

## Usage

```typescript
import { } from '@appgen/codegen-provenance';
```

## Development

```bash
pnpm -F @appgen/codegen-provenance build
pnpm -F @appgen/codegen-provenance test
```
