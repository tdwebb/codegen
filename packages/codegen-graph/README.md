# @appgen/codegen-graph

codegen-graph package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-graph
```

## Usage

```typescript
import { } from '@appgen/codegen-graph';
```

## Development

```bash
pnpm -F @appgen/codegen-graph build
pnpm -F @appgen/codegen-graph test
```
