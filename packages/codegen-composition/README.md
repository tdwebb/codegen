# @appgen/codegen-composition

codegen-composition package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-composition
```

## Usage

```typescript
import { } from '@appgen/codegen-composition';
```

## Development

```bash
pnpm -F @appgen/codegen-composition build
pnpm -F @appgen/codegen-composition test
```
