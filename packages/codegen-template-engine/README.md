# @appgen/codegen-template-engine

codegen-template-engine package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-template-engine
```

## Usage

```typescript
import { } from '@appgen/codegen-template-engine';
```

## Development

```bash
pnpm -F @appgen/codegen-template-engine build
pnpm -F @appgen/codegen-template-engine test
```
