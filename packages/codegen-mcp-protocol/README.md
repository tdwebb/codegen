# @appgen/codegen-mcp-protocol

codegen-mcp-protocol package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-mcp-protocol
```

```typescript
import { } from '@appgen/codegen-mcp-protocol';
```

## Development

```bash
pnpm -F @appgen/codegen-mcp-protocol build
pnpm -F @appgen/codegen-mcp-protocol test
```
