# @appgen/codegen-diff

codegen-diff package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-diff
```

## Usage

```typescript
import { } from '@appgen/codegen-diff';
```

## Development

```bash
pnpm -F @appgen/codegen-diff build
pnpm -F @appgen/codegen-diff test
```
