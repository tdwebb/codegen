# @appgen/codegen-registry

codegen-registry package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-registry
```

## Usage

```typescript
import { } from '@appgen/codegen-registry';
```

## Development

```bash
pnpm -F @appgen/codegen-registry build
pnpm -F @appgen/codegen-registry test
```
