# @appgen/codegen-validator

codegen-validator package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-validator
```

## Usage

```typescript
import { } from '@appgen/codegen-validator';
```

## Development

```bash
pnpm -F @appgen/codegen-validator build
pnpm -F @appgen/codegen-validator test
```
