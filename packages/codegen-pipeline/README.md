# @appgen/codegen-pipeline

codegen-pipeline package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-pipeline
```

## Usage

```typescript
import { } from '@appgen/codegen-pipeline';
```

## Development

```bash
pnpm -F @appgen/codegen-pipeline build
pnpm -F @appgen/codegen-pipeline test
```
