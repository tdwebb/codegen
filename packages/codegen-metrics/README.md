# @appgen/codegen-metrics

codegen-metrics package for the CodeGen platform.

## Installation

```bash
pnpm install @appgen/codegen-metrics
```

## Usage

```typescript
import { } from '@appgen/codegen-metrics';
```

## Development

```bash
pnpm -F @appgen/codegen-metrics build
pnpm -F @appgen/codegen-metrics test
```
